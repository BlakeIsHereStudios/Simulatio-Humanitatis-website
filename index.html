Simulatio Humanitatis

What the hell is this?

Simulatio Humanitatis is an open-source in-development 4X turn-based strategy game built with Godot, inspired by the Civilization series.

By the way, "Simulatio Humanitatis" means "Simulation of Humanity" in Latin. It's a working title as of now, though.

So, when is it coming out?

Development takes a while, but a working build will eventually be available here. If you're tech-savvy and want to mess with the code or compile it yourself, a GitHub repository will be made as well.

Join the dev team!

This is currently a one-man project, but I do want more people to work on it. If you want to join the team, then make a pull request asking to join the team on the GitHub repository (when it comes).
